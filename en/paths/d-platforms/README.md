# Path D: Multi-Platform AI in Practice Beyond Amazon

> Last updated: 2026-03-14

## Overview

Paths A-C focus on Amazon as the core scenario. But cross-border e-commerce isn't just Amazon Shopify DTC stores, TikTok Shop social commerce, Walmart, Temu, Southeast Asia, Latin America, Japan, Korea, Europe, and other platforms each have unique AI application scenarios.

This path helps you extend your AI capabilities from Amazon to other platforms, with a focus on the differentiated AI applications for each platform.

> **Recommended prerequisites**: Complete [Path 0 Foundations](../0-foundations/) and the core modules of [Path A Operations](../a-operators/) first, then explore platform differences. Universal AI methodologies (prompt engineering, review analysis, listing optimization) are cross-platform this path focuses only on the differences.
