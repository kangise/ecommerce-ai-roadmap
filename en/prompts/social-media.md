## 1. Instagram Reels Script Batch Generation

> Source: [E1 Instagram + Facebook](../paths/e-social-media/e1-instagram-facebook-ai-guide.md)

```
You are an Instagram Reels creative expert specializing in e-commerce brand content.

Product information:
- Product name: [Name]
- Brand positioning: [Premium / Mid-range / Value]
- Core selling points: [3 points]
- Price: $[X]
- Target audience: [Age, gender, lifestyle]

Please generate 5 Reels scripts from different angles, each including:
1. Visual hook description (first 2 seconds of footage)
2. Shot-by-shot script (visuals + duration + text overlay for each shot)
3. Recommended background music style
4. Caption copy (with hashtag strategy)
5. CTA design

The 5 angles are:
- Angle 1: Lifestyle scene (aspirational)
- Angle 2: Before/After comparison
- Angle 3: Educational ("X reasons to choose this product")
- Angle 4: Trend-riding (adapted to current popular Reels formats)
- Angle 5: UGC style (simulating real user sharing)

Requirements: Each script 15-30 seconds, Instagram style: polished, design-forward, not overly salesy.
```

## 2. Meta Advantage+ Ad Copy Generation

> Source: [E1 Instagram + Facebook](../paths/e-social-media/e1-instagram-facebook-ai-guide.md)

```
You are a Meta Ads copywriting expert who specializes in high-converting e-commerce ads.

Product information:
- Product: [Name]
- Core selling points: [3 points]
- Price: $[X] (original $[X], XX% off)
- Target audience: [Age, gender, interests, pain points]
- Landing page: [Shopify product page / Amazon Listing]

Please generate 5 sets of ad copy, each including:
1. Primary Text (body copy, 3 versions: short ≤125 characters / medium ≤250 characters / long ≤500 characters)
2. Headline (≤40 characters)
3. Description (≤30 characters)
4. CTA button recommendation (Shop Now / Learn More / Get Offer)

5 angles: Pain point, Social proof, Limited-time offer, Product features, Emotional connection.
```

## 3. YouTube Product Review Script

> Source: [E2 YouTube](../paths/e-social-media/e2-youtube-ai-guide.md)

```
You are a YouTube product review video script expert.

Product information:
- Product name: [Name]
- Brand: [Brand]
- Price: $[X]
- Core features: [List 5]
- Main competitors: [Competitor 1], [Competitor 2]
- Amazon review summary: Positive reviews focus on [X], negative reviews focus on [X]

Please generate a 10-12 minute review video script with this structure:
1. Hook (0:00-0:30) — One-sentence summary + suspense
2. Unboxing & appearance (0:30-2:00)
3. Feature testing (2:00-6:00) — Test each feature, score 1-10
4. Pros & cons summary (6:00-8:00)
5. Who it's for / not for (8:00-9:30)
6. Conclusion & CTA (9:30-10:30)

For each section provide: voiceover script, visual suggestions, chapter marker timestamps.
```

## 4. Xiaohongshu Seeding Post

> Source: [E3 Xiaohongshu](../paths/e-social-media/e3-xiaohongshu-ai-guide.md)

```
You are a Xiaohongshu (Little Red Book) viral post creation expert. Your writing style is authentic, conversational, like a friend sharing a great find.

Product information:
- Product name: [Name]
- Category: [X]
- Price: [X] yuan
- Core selling points: [3 points]
- Target audience: [Age, scenario, pain points]

Please generate 3 seeding posts from different angles, each including:
1. Cover title (max 20 characters, include numbers or pain points)
2. Body text (300-500 words, first person, conversational, with emoji)
3. Hashtag strategy (15-20: trending + category + long-tail)
4. Cover image suggestions

3 angles: Pain point solution, Scenario seeding, Comparison review.
Do not use absolute terms like "best," "number one," or "absolutely" (violates advertising regulations).
```

## 5. Cross-Platform Content Adaptation

> Source: [E7 Cross-Channel Coordination](../paths/e-social-media/e7-social-media-cross-channel.md)

```
You are a cross-platform content adaptation expert.

Here is the core content from a product review:
[Paste core script/copy]

Please adapt it for the following platforms:
1. YouTube long-form video description (with SEO keywords + chapter markers)
2. YouTube Shorts scripts (3 clips, each 30-60 seconds)
3. Instagram Reels scripts (2, 15-30 seconds, polished aesthetic style)
4. Instagram Carousel copy (8 slides)
5. TikTok scripts (2, 15-60 seconds, entertainment/hook style)
6. Pinterest Pin titles + descriptions (5 different angles)
7. Xiaohongshu seeding post (1, 300-500 words, conversational)

Adjust tone, length, and CTA for each platform to match its style.
```
