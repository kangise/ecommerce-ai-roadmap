## How to Use

1. Click a file link in the table above to find the template you need
2. Copy the text inside the Prompt code block
3. Paste into ChatGPT / Claude / Gemini, and replace the placeholders in `[brackets]`
4. After getting results, use the "Tips" to ask follow-up questions for refinement
