# Path C: 管理者 AI 战略落地

> 最后更新: 2026-03-11

## 概述

- **目标受众**: 团队负责人/创始人
- **前提条件**: 不需要技术背景，但需要对业务有深入理解
- **时间投入**: 集中 3-5 小时完成评估和规划
- **核心产出**: 一份 AI 落地规划书

> 理解 AI 能为团队做什么，制定可执行的 AI 落地计划

## 进度追踪

```
[x] C1. 评估：完成团队 AI 能力评估，输出优先级排序
[x] C2. 建设：团队至少 80% 的人每天在使用 AI 工具
[x] C3. ROI：完成至少一个 AI 项目的 ROI 评估报告
```

**Path C 总完成标志：** 输出一份完整的团队 AI 落地规划书（包含优先级、时间线、预算、KPI）。
