# Path A: 运营人 AI 提效实战

> 最后更新: 2026-03-10

## 概述

- **目标受众**: 选品/运营/广告/客服岗位的电商从业者
- **前提条件**: 有基本的电商运营经验（知道什么是 ASIN、PPC、FBA）
- **时间投入**: 每天 30 分钟，2-4 周完成全部模块
- **核心产出**: 一套可复用的 AI 工作流

> 不写一行代码，用 AI 工具把日常运营效率提升 3-10 倍

## 进度追踪

