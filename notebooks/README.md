## 如何使用 | How to Use

1. 点击 Notebook 右侧的 **Open in Colab** 按钮
2. 在 Colab 中点击 **Runtime → Run all** 运行所有单元格
3. 所有依赖会在第一个代码单元格自动安装

> 所有 Notebook 均在 Google Colab 免费版测试通过，无需 GPU。
