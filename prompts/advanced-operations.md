## 1. AI 产品图片生成（A7）

> 详细方法论: [A7 视觉内容](../paths/a-operators/a7-visual-content.md)

```
你是一个电商产品图片策划专家。

我的产品：[名称]
品类：[X]
目标平台：[Amazon/Shopify/Instagram]
品牌色：[X]

请为这个产品规划一套完整的图片方案：

1. Amazon 主图（白底，产品占 85%+）
- 拍摄角度建议
- 光照方案
- AI 生成 Prompt（Midjourney/DALL-E 格式）

2. 场景图 x3
- 3 个不同使用场景的描述
- 每个场景的 AI 生成 Prompt

3. 信息图 x2
- 尺寸对比图的布局建议
- 功能亮点图的布局建议

4. A+ Content 图 x3
- 品牌故事图
- 产品对比图
- 使用教程图

5. 社交媒体素材
- Instagram 正方形图 Prompt
- TikTok 竖版图 Prompt

每个 Prompt 包含正面描述和负面排除词。
```

**使用技巧**: 生成后用 rembg（Python）去背景，再用 Pillow 合成白底。详见 [B9 AI 图片 Pipeline](../paths/b-developers/b9-ai-image-pipeline.md)。

## 2. 竞品价格监控与动态定价（A8）

> 详细方法论: [A8 定价策略](../paths/a-operators/a8-pricing-strategy.md)

```
你是一个电商定价策略专家。

我的产品：[名称]
当前售价：$[X]
成本（含 FBA）：$[X]
当前利润率：[X]%

竞品价格数据（过去 30 天）：
- 竞品 A：$[X] → $[X]（变化 [X]%）
- 竞品 B：$[X]（稳定）
- 竞品 C：$[X] → $[X]（变化 [X]%）

请分析：
1. 竞品价格变化的可能原因
2. 我应该跟进调价还是保持不变？
3. 如果调价，最优价格点是多少？（考虑利润率和销量弹性）
4. 大促期间（Prime Day/BFCM）的定价策略
5. 长期定价路线图（3-6 个月）
```

## 3. GEO 审计与 AI 搜索优化（A9）

> 详细方法论: [A9 SEO/GEO](../paths/a-operators/a9-seo-geo.md)

```
你是一个 GEO（Generative Engine Optimization）专家。

我的品牌：[名称]
产品：[X]
网站：[URL]（如果有 Shopify）
Amazon ASIN：[X]

请做 GEO 审计：

1. 结构化数据检查
- Product Schema 是否完整？
- FAQ Schema 是否存在？
- Review Schema 是否存在？

2. AI 搜索可见度测试
- 在 ChatGPT 搜索 "best [品类] 2026"，我的品牌是否被提及？
- 在 Perplexity 搜索，是否被推荐？
- 竞品 [A/B/C] 是否被推荐更多？

3. 优化建议
- 需要添加/修复的 Schema
- 需要创建的 FAQ 内容（10 个问题）
- 需要获取的第三方引用/评测
- Agentic Commerce 准备度评估

4. 优先级行动清单（1 周 / 1 月 / 3 月）
```

## 4. 品牌故事与视觉系统生成（A10）

> 详细方法论: [A10 品牌建设](../paths/a-operators/a10-brand-building.md)

```
你是一个品牌策略专家，擅长为跨境电商品牌创建品牌体系。

品牌信息：
- 品牌名：[名称]
- 品类：[X]
- 目标市场：[US/EU/JP]
- 目标客户：[年龄/性别/生活方式]
- 与竞品的差异：[你的独特之处]
- 创始人背景：[简述]

请一次性生成完整品牌体系：

1. 品牌故事（200-300 字，适合 Amazon Brand Store / Shopify About）
2. 品牌使命（1 句话，≤30 字）
3. 品牌 Tagline（3 个选项，每个 ≤8 词）
4. 品牌价值观（3-5 个）
5. 品牌声音指南（各平台语气规范）
- Amazon Listing 语气
- Shopify 产品页语气
- Instagram 语气
- TikTok 语气
6. 配色方案建议（主色+辅色+功能色，含色号）
7. Amazon A+ Content 品牌故事模块文案
```

## 5. SKU 真实利润计算（A11）

> 详细方法论: [A11 财务分析](../paths/a-operators/a11-financial-analysis.md)

```
你是一个跨境电商财务分析专家。

产品数据（过去 30 天）：
- 产品：[名称]
- 售价：$[X]，月销量：[X] 单
- 采购成本（FOB）：$[X]/件
- 国际运费：$[X]/件
- 关税：[X]%
- Amazon 佣金：[X]%
- FBA 配送费：$[X]/件
- FBA 月仓储费：$[X]/件
- 广告花费：$[X]/月
- 退货率：[X]%
- 工具订阅：$[X]/月

请计算：
1. 单件真实利润（扣除所有成本，含隐藏成本）
2. 单件利润率
3. 月度总利润
4. 盈亏平衡点（需要卖多少件覆盖固定成本）
5. 成本结构饼图（哪项占比最高）
6. 3 个降低成本的具体建议
7. 售价 ±10% 的利润敏感度分析
```

## 6. IP 风险全面评估（A12）

> 详细方法论: [A12 知识产权](../paths/a-operators/a12-ip-protection.md)

```
你是一个知识产权风险评估专家。

我计划销售的产品：
- 品类：[X]
- 核心功能：[列出 3-5 个]
- 外观特征：[描述]
- 目标市场：[US/EU/JP]

请做全面 IP 风险评估：

1. 专利风险（高/中/低）
- 这个品类常见的专利类型
- 需要排查的关键专利数据库
- 高风险功能/设计特征

2. 商标风险（高/中/低）
- 品牌名是否可能与已有商标冲突
- 建议的商标检索步骤

3. 版权风险（高/中/低）
- 产品图片/文案的版权注意事项
- AI 生成内容的版权政策

4. TRO 风险（高/中/低）
- 这个品类是否有频繁的 TRO 案件
- 预防措施

5. 总体风险评级和建议
- Go / Caution / No-Go 决策
- 如果 Go：需要先完成的合规步骤
```

## 7. AI Growth Hack 选品验证（A13）

> 详细方法论: [A13 AI Growth Hack](../paths/a-operators/a13-ai-growth-hack.md)

```
你是一个跨境电商选品专家，精通 AI 数据分析。

我的条件：
- 启动资金：$[X]
- 目标市场：[US/EU/JP]
- 供应链：[中国工厂/1688/贸易商]
- 运营经验：[新手/1-2年/3年+]

请用 AI Growth Hack 框架帮我选品：

1. 推荐 5 个品类（标注市场规模/竞争度/利润率/入门门槛）
2. 每个品类的 Review 痛点分析（用户最常抱怨什么 = 差异化机会）
3. 最终推荐 1 个品类 + 完整理由
4. 差异化策略
5. 预估首批订单量和启动成本
6. 6 个月 ROI 预测
7. 从选品到上架的 AI 加速时间线
```

## 8. 每日 AI 运营晨报（A13）

> 详细方法论: [A13 AI Growth Hack](../paths/a-operators/a13-ai-growth-hack.md#7-ai-agent-工作流实战)

```
你是我的电商运营 AI 助手。请基于以下数据生成今日运营晨报。

昨日数据：
- 总收入：$[X]（vs 前天 $[X]）
- 总订单：[X] 单
- 广告花费：$[X]，广告销售：$[X]
- ACOS：[X]%
- 退货：[X] 单
- 新差评：[X] 条

请生成：
1. 关键指标摘要（收入/订单/ACOS 的日环比和周环比）
2. 异常告警（哪些指标偏离正常范围）
3. 今日优先行动清单（按 P0/P1/P2 排序）
4. 一条优化建议（基于数据趋势）

格式简洁，适合 5 分钟内阅读。
```
