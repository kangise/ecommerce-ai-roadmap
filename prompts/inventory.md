## 1. 安全库存计算

```
你是一个跨境电商库存管理专家。请帮我计算以下产品的安全库存水平。

产品信息：
- ASIN：[ASIN]
- 日均销量：[X] 件（过去 30 天平均）
- 销量波动：日标准差 [X] 件
- 供应商交期：[X] 天（从下单到入仓）
- 交期波动：±[X] 天
- 目标服务水平：[95%/99%]

请输出：
1. 安全库存数量（件）及计算过程
2. 再订货点（ROP）
3. 建议订货批量（EOQ 估算）
4. 预计断货风险评估
```

## 2. 补货决策分析

```
你是一个 Amazon FBA 库存规划师。请基于以下数据给出补货建议。

当前库存状态：
- FBA 可售库存：[X] 件
- FBA 在途库存：[X] 件（预计 [X] 天后到达）
- 海外仓库存：[X] 件
- 工厂在产：[X] 件（预计 [X] 天后完工）

销售趋势：
- 过去 7 天日均：[X] 件
- 过去 30 天日均：[X] 件
- 是否有即将到来的大促：[是/否，日期]
- 季节性因素：[淡季/旺季/平稳]

请输出：
1. 当前库存可售天数
2. 是否需要立即补货（是/否 + 理由）
3. 建议补货数量和时间节点
4. 大促备货建议（如适用）
5. 风险提示（断货/滞销/仓储费）
```

## 3. 滞销库存处理

```
你是一个库存优化顾问。以下产品出现滞销，请给出处理方案。

滞销产品信息：
- ASIN：[ASIN]
- 当前库存：[X] 件
- 过去 30 天销量：[X] 件
- 库龄分布：0-90天 [X]件 / 91-180天 [X]件 / 181-270天 [X]件 / 271-365天 [X]件
- 产品成本：$[X]/件
- 当前售价：$[X]
- 月仓储费：$[X]

请输出：
1. 滞销严重程度评估（轻度/中度/重度）
2. 处理方案优先级排序：
- 降价促销（建议价格和预期清货周期）
- Outlet/Warehouse Deals
- 移除到海外仓
- 销毁（成本对比分析）
3. 每个方案的成本收益分析
4. 推荐方案及执行时间线
```
