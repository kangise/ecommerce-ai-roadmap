## 如何使用

1. 点击上方表格中的文件链接，找到你需要的模板
2. 复制 Prompt 代码块中的文本
3. 粘贴到 ChatGPT / Claude / Gemini，替换 `[方括号]` 中的占位符
4. 获得结果后，可根据"使用技巧"追问优化
